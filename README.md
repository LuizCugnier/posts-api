# API REST 

## Descrição

API RESTful feita com nestjs para criação de posts com autenticação de usuário.

## Instalação

#### Instalar as dependências 
```bash
$ pnpm install
```
### Inicializar o docker
```
$ docker-compose up -d
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
